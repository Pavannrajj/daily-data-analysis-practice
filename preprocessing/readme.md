# Exploring and Pre-Processing Data Using Pandas

This repository contains a practice notebook focused on **data exploration and preprocessing** using the Python **Pandas** library.  
The goal is to understand the dataset structure, clean the data, and prepare it for further analysis or visualization.

---

## Objectives

- Load and explore the dataset
- Understand data types, shape, and summary statistics
- Identify and handle missing values
- Clean and preprocess data
- Prepare the dataset for visualization or machine learning models

---

## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook
